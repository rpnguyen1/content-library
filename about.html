About This Project

This is a demo content library for template creators. I know finding footage for templates is often very time consuming. So I created this to help make it easier for template creators to find footage for their templates. All footage is copyright free and can be used in commercial projects without attribution. It has a search function, type filter, and category filter to help you find footage quickly. If you have any suggestions or feedback, please let me know!
